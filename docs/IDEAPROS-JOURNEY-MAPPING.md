# IdeaPros 120-Step Journey Mapping to Sahara Oases

**Linear:** AI-1802
**Date:** 2026-03-09
**Source:** IdeaPros ~120-step founder program (Fred Cary), mapped to Sahara's 5 Oases stages.

## Overview

IdeaPros guides founders from raw idea to investment-ready company through ~120 discrete steps. This document maps each step to one of Sahara's 5 Oases stages with clear completion criteria. Steps are grouped into sub-milestones within each stage.

The 5 Oases stages: **Clarity** -> **Validation** -> **Build** -> **Launch** -> **Grow**

Fred Cary's 9-Step Startup Process provides the backbone:
1. Define the Real Problem
2. Identify the Buyer and Environment
3. Establish Founder Edge
4. Define the Simplest Viable Solution
5. Validate Before Building
6. Define the First Go-To-Market Motion
7. Install Execution Discipline
8. Run a Contained Pilot
9. Decide What Earns the Right to Scale

## Completion Check Types

Each step uses one of these automated or manual check types:
- `profile_field` -- Checks a specific profile field is populated
- `reality_lens_done` -- Reality Lens assessment completed
- `chat_sessions` -- Minimum chat session count with FRED
- `document_created` -- Strategy/plan document exists
- `pitch_deck_uploaded` -- Pitch deck uploaded and scored
- `investor_readiness_scored` -- IRS assessment completed with minimum score
- `metric_threshold` -- A tracked metric meets a numeric threshold
- `boardy_ready` -- Boardy investor matching profile complete
- `manual` -- FRED or user marks as complete during coaching
