## Objective

Create an Investor Readiness Score (IRS) system that:
1. Evaluates startup across 6 key categories
2. Produces a 0-100% overall score
3. Provides specific, actionable feedback
4. Tracks improvement over time
5. Integrates with uploaded documents

## IRS Categories and Weights

| Category | Weight | What It Measures |
|----------|--------|------------------|
| Team | 25% | Founder experience, team completeness, relevant expertise |
| Market | 20% | TAM/SAM/SOM clarity, market timing, competitive landscape |

| Traction | 15% | Revenue, users, growth rate, key metrics |

## Implementation Steps

### Step 1: Create IRS Scoring Schema

**File:** `lib/db/migrations/010_investor_readiness.sql`

```sql
-- IRS assessments
CREATE TABLE investor_readiness_scores (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  overall_score DECIMAL(5,2) NOT NULL,
  category_scores JSONB NOT NULL,
  -- { team: 75, market: 80, product: 65, traction: 50, financials: 60, pitch: 70 }
  strengths TEXT[] DEFAULT '{}',
  weaknesses TEXT[] DEFAULT '{}',
  recommendations JSONB DEFAULT '[]',
  source_documents UUID[] DEFAULT '{}', -- Documents used in assessment
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_irs_user ON investor_readiness_scores(user_id);
CREATE INDEX idx_irs_created ON investor_readiness_scores(created_at DESC);
```

### Step 2: Create IRS Scoring Engine

**File:** `lib/fred/investor-readiness.ts`

```typescript
interface IRSInput {
  startupInfo: StartupInfo;
  documents?: Document[];
  previousScore?: IRSScore;
}

interface IRSScore {
  overall: number;
  categories: {
    team: CategoryScore;
    market: CategoryScore;
    product: CategoryScore;
    traction: CategoryScore;
    financials: CategoryScore;
    pitch: CategoryScore;
  };
  strengths: string[];
  weaknesses: string[];
  recommendations: Recommendation[];
}

interface CategoryScore {
  score: number;
  confidence: number;
  factors: ScoringFactor[];
```

## Scoring Logic

### Team Score (25%)
```
Base: 50
+ 10 if founder has relevant industry experience
+ 10 if technical co-founder present
+ 10 if previous startup experience
+ 5 per filled key role (up to 20)
- 10 if solo founder with no advisors
- 5 per critical unfilled role
```

### Market Score (20%)
```
Base: 50
+ 15 if TAM > $1B and well-defined
+ 10 if clear SAM/SOM path
+ 10 if timing is favorable
+ 5 if competitive landscape mapped
- 10 if market shrinking
- 15 if no clear differentiation
```
