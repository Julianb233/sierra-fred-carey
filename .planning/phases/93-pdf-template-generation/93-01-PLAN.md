  <name>Task 2: Trigger.dev PDF generation task — trigger/generate-founder-report-pdf.ts</name>

  <action>
Create `trigger/generate-founder-report-pdf.ts` that exports `generateFounderReportPdf` as a Trigger.dev task.

**Import pattern (match existing codebase):**
```typescript
import { task, logger } from "@trigger.dev/sdk/v3";
```

**Task definition:**
```typescript
export const generateFounderReportPdf = task({
  id: "generate-founder-report-pdf",
  maxDuration: 120,
  retry: {
    maxAttempts: 2,
    minTimeoutInMs: 5000,
    maxTimeoutInMs: 30000,
    factor: 2,
  },
  run: async (payload: { reportId: string; userId: string; version: number; reportData: ReportData }) => { ... }
});
```

**Inside `run()` — sequential steps:**

1. Record start time (`Date.now()`)
2. `await updateReportStatus(reportId, { status: "generating" })` — mark as generating
3. Build the PDF document: `const doc = buildReportDocument(reportData)` (import from `@/lib/report/pdf-template`)
4. Render to buffer: `const buffer = await renderToBuffer(doc)` (import `renderToBuffer` from `@react-pdf/renderer`)
5. Log buffer size: `logger.log("PDF rendered", { bytes: buffer.byteLength })`
6. Upload to Vercel Blob:
   ```typescript
   import { put } from "@vercel/blob";
   const blobPath = `founder-reports/${userId}/v${version}-report.pdf`;
   const blob = await put(blobPath, buffer, {
     access: "public",
     addRandomSuffix: false,
     contentType: "application/pdf",
   });
   ```
7. Calculate elapsed ms: `const generationMs = Date.now() - startTime`
8. Update DB: `await updateReportStatus(reportId, { status: "complete", pdfBlobUrl: blob.url, pdfSizeBytes: buffer.byteLength, generationMs })`
9. Log completion: `logger.log("PDF generation complete", { url: blob.url, sizeBytes: buffer.byteLength, generationMs })`
10. Return `{ pdfUrl: blob.url, sizeBytes: buffer.byteLength, generationMs }`

**Error handling:**
Wrap the entire run body in try/catch. On error:
- `logger.error("PDF generation failed", { reportId, error: String(error) })`
- `await updateReportStatus(reportId, { status: "failed" })` (best-effort, don't re-throw from this)
- Re-throw the original error so Trigger.dev retry kicks in

**Imports needed:**
- `task, logger` from `@trigger.dev/sdk/v3`
- `renderToBuffer` from `@react-pdf/renderer`
- `put` from `@vercel/blob`
- `updateReportStatus` from `@/lib/db/founder-reports`
- `buildReportDocument` from `@/lib/report/pdf-template`
- `ReportData` type from `@/types/report`

**Do NOT:**
- Use `triggerAndWait` — API route returns 202, client polls
- Use `schedules.task` — this is on-demand, not scheduled
- Pass stepSnapshot in payload (too large) — only pass reportData
- Call Font.register inside run() — it's already at module level in pdf-template.ts
